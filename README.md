# rickandmortyapi-db

Generates an SQLite database from [The Rick and Morty API](https://rickandmortyapi.com).


## How to

```shell
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip3 install -r requirements.txt
$ python3 rickandmortyapi.py
```
